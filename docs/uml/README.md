# UML 图文件存放目录

请将以下 UML 图文件放置于此目录（或子目录），便于版本管理与答辩展示：

- **用例图**：如 `usecase.puml` 或 `usecase.png`
- **类图**：如 `class.puml` 或 `class.png`
- **顺序图**：如 `sequence-stock-in.puml`、`sequence-purchase.puml`
- **活动图/状态图**：如 `activity-approval.puml`、`state-prescription.puml`

可使用 PlantUML、Draw.io、StarUML 等工具绘制，导出为 `.puml` 或图片格式。
